# 🏨 Hotel Reservation System

A console-based application that allows users to manage hotel rooms and handle customer bookings using Java and MySQL.

---

## 🚀 Features

- Add new rooms
- View available rooms
- Book a room for customers
- Store booking details (name, room, check-in, check-out)
- Database connectivity using JDBC

---

## 🛠️ Tech Stack

- Java (Core Java)
- JDBC
- MySQL
- Eclipse IDE

---

## 🗄️ Database Tables

### Room Table
- id
- roomNumber
- roomType
- price
- status

### Booking Table
- id
- customerName
- roomId
- checkIn
- checkOut
